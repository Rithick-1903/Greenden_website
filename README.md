## Greenden Flowers Sale Website

This project is a simple flower sale website called **Greenden**, built using **HTML** and **Tailwind CSS**. It features a clean and modern design for showcasing flower products, special offers, and a sale banner section.

The website includes a responsive layout, product display cards, and a visually appealing interface created using Tailwind utility classes. The main goal of this project is to practice building fast and responsive UI designs using Tailwind CSS.

This project focuses only on the front-end design and does not include backend functionality or real-time data.

### Technologies Used

* HTML5
* Tailwind CSS

### Features

* Responsive flower shop layout
* Product cards with images and details
* Sale/offer banner section
* Clean and modern UI design
